# Cascade Complete Cleaning Website

Static one-page website for Cascade Complete Cleaning Services Inc.

## Files

- `index.html` - page structure, SEO metadata, schema, content
- `style.css` - responsive styling
- `script.js` - quote form, chat widget, steam animation

## Important

The chat widget is a safe static assistant. It does not use an AI API key in the browser because that would be insecure and would break on a static GitHub site. To make it a real AI chatbot, add a backend endpoint later through Netlify Functions, Vercel Functions, or another server.
